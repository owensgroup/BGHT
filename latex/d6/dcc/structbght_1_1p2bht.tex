P2\+BHT P2\+BHT (power-\/of-\/two bucketed hash table) is an associative static GPU hash table that contains key-\/value pairs with unique keys. The hash table is an open addressing hash table based on the power-\/of-\/two hashing to balance loads between buckets (bucketed and using two hash functions).

The hash table is an open addressing hash table based on the power-\/of-\/two hashing to balance loads between buckets (bucketed and using two hash functions).
