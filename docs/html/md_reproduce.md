# []{#autotoc_md17 .anchor} Running the benchmarks

After building you should have the following binaries

::: fragment
::: line
user:\~/github/bght/build/bin\$ ls
:::

::: line
probes_per_technique rates_per_technique rates_per_technique_fixed_lf
:::
:::

1.  `probes_per_technique`: An experiment to evaluate the average number
    of probes per key for a constant number of keys and all different
    probing schemes and load factors\
2.  `rates_per_technique`: An experiment to evaluate the average
    insertion and find throughputs for a constant number of keys and all
    different probing schemes and load factors\
3.  `rates_per_technique_fixed_lf`: An experiment to evaluate the
    average insertion and find throughputs for two constant load
    factors, a range of keys, and all different probing schemes\

All executables have CPU validation that can be turned on by using the
flag `--validate=true`. Additional options are shown below:

::: fragment
::: line
./probes_per_technique
:::

::: line
\--num-keys Number of keys for all experiments e.g, \--num-keys=512
:::

::: line
./rates_per_technique
:::

::: line
\--num-keys Number of keys for all experiments e.g, \--num-keys=512
:::

::: line
./rates_per_technique_fixed_lf
:::

::: line
\--min-keys Minimum number of keys for all experiments e.g,
\--min-keys=512
:::

::: line
\--max-keys Maximum number of keys for all experiments e.g,
\--max-keys=1024
:::

::: line
\--step Step between the minimum and maximum for all experiments e.g,
\--step=32
:::

::: line
\--load-factor1 First constant load factor for all experiments e.g,
\--load-factor1=0.8
:::

::: line
\--load-factor2 First constant load factor for all experiments e.g,
\--load-factor1=0.9
:::
:::

# []{#autotoc_md18 .anchor} Plotting the results

A Python script is under `bght/figs` which will parse all outputs of the
benchmarking step and reproduce the figures (in SVG format) in the
paper. The script requires the packages: pandas, Matplotlib, NumPy, and
SciPy. The script takes the following arguments:

::: fragment
::: line
python ./plot.py
:::

::: line
-d Root directory that contains the results
:::

::: line
-mf Minimum find rate for the charts axes (optional)
:::

::: line
-xf Maximim find rate for the charts axes (optional)
:::

::: line
-mi Minimum insertion rate for the charts axes (optional)
:::

::: line
-xi Maximim insertion rate for the charts axes (optional)
:::

::: line
-p Specify the probing scheme to plot results for (optiona - default
plots all probing schemes)
:::
